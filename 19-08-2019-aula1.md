### Comandos GIT:
* Para inicializar um repositório git
```bash
git init
```
* Adicionar novos arquivos ou mudanças feitas no git
  * utilizar "." para adicionar todos os arquivos ou especificar o nome do arquivos para adicionar apenas 1
```bash
git add .
```
